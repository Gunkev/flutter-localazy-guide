This repository contains the starter and completed Flutter projects for a Localazy localization guide.

- `flutter-demo` is the starter app.
- `flutter-localazy-demo` is the completed app with ARB files, Localazy CLI config, translated strings, and locale switching.

Use `flutter-demo` if you want to follow the guide step by step. Use `flutter-localazy-demo` if you want to inspect or run the finished implementation.
